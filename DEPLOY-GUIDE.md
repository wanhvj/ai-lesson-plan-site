# AI教案网站上线指南

## 现在这套最适合你的上线方式

这套项目已经整理成了“一个地址对外访问”的形式：

- 前端页面和后端接口走同一个线上地址
- 适合直接挂一个正式域名
- 对新手最省事

推荐路线：

1. 把代码放到 GitHub
2. 用 Render 部署
3. 先拿到一个可公开访问的网址
4. 再买域名并绑定

## 一、上线前确认

项目关键目录：

- 前端：`apps/web/client`
- 后端：`apps/server`
- Render 配置：`render.yaml`

## 二、上传到 GitHub

1. 新建一个 GitHub 仓库
2. 把当前项目推上去
3. 确认仓库里包含：
   - `apps/server`
   - `apps/web/client`
   - `render.yaml`

## 三、用 Render 部署

1. 打开 Render
2. 用 GitHub 登录
3. 选择 `New +`
4. 选择 `Blueprint`
5. 选中这个项目仓库
6. Render 会识别 `render.yaml`
7. 直接创建服务

部署成功后，你会拿到一个：

- `https://xxx.onrender.com`

这个地址就是你的网站公网地址。

## 四、绑定正式域名

1. 先购买一个域名
2. 回到 Render 服务设置
3. 找到 `Custom Domains`
4. 添加你的域名
5. 按 Render 提示去域名平台配置 DNS
6. 等验证通过

## 五、关于备案

如果你把网站部署在海外或中国香港服务器：

- 一般不需要备案

如果你后面改成中国大陆服务器：

- 就需要备案后再正式用域名访问

## 六、你今天最适合的实际顺序

1. 先把项目上传 GitHub
2. 先在 Render 跑通公网地址
3. 公网地址没问题后再买域名
4. 域名绑好后再考虑要不要做备案版
